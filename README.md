# `Bun.js` api using DDD

## A basic example how to use `Bun.js` to build apis using npm packages

## First steps

Install `Bun.js` with `asdf`

```bash
asdf plugin add bun
```

```bash
asdf install bun latest
```

## How to test

Running the application

```bash
bun run dev
```
